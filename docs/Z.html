(static) print(dataframe) → {String}

Source:
  • zebras.js, line 88

Prints dataframe.

Returns the entire dataframe as an ASCII table. If working in a local Node environment, wrap this and other printing functions in console.log() to display ASCII tables.

Z.print(df)
